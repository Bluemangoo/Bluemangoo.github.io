Explorer Patcher 推荐

简介

Windows 11 的任务栏和右键菜单令人厌恶，对吧。

这个工具可以让任务栏变回你熟悉的样子。

功能简述

  • 在 Windows 11 或 Windows 10 任务栏之间进行选择（支持标签、小图标和大量自定义）。
  • 禁用 Windows 11 文件资源管理器的上下文菜单和命令栏。
  • 打开开始菜单时直接打开“所有应用程序”（默认）、选择要显示的常用应用程序的数量、在活动监视器上显示等等。
  • 选择 Windows 11、Windows 10 或 Windows NT Alt-Tab 窗口切换器（Alt+Tab）。
  • 外壳（Shell）的许多质量改进，例如：
    • 使托盘菜单与 Windows 风格相匹配，使它们表现得像浮出控件，并将它们相对于图标居中。
    • 选择左键、右键单击网络图标时的操作。
    • 恢复到 Windows 7 文件资源管理器搜索框，或完全禁用 Windows 搜索。
    • 禁用沉浸式菜单并使用缓解措施来帮助您运行真正的经典主题而不会出现故障。

安装和卸载

运行ep_setup.exe安装

卸载的三种方式

  1. 在appwiz.cpl（或控制面板-程序-程序与功能）中找到ExplorerPatcher选择卸载
  2. 运行ep_setup.exe /uninstall
  3. 将ep_setup.exe重命名为ep_uninstall.exe并运行
